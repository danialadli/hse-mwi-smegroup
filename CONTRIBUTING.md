# Contributing to HSE-MWI
Thank you for your interest in contributing to our project. We truly appreciate your time and effort to help improve the Mental Wellness Index tool.
To ensure a smooth collaboration, we have outlined a few guidelines below.

## Getting Started
To run the application locally on your machine, please follow these steps:

1. Open `app.R` in RStudio.
2. Run `renv::restore()` to install dependencies.
3. Click the "Run App" button at the top of the editor.

## Reporting Bugs or Issues
Found a bug? We would love to hear about it. Please use the GitHub Issues tab.
- Be as specific as possible.
- If you can, include screenshots or the error message to help us understand the problem better.
